# Music School Database Management System

## Overview

Designed and implemented a relational database system for managing a music school. The system supports student enrollment, lesson scheduling, instructor management, room bookings, and course materials.

## Technologies

* PostgreSQL
* SQL
* PL/pgSQL
* Python
* Jupyter Notebook

## Key Tasks

* Designed and implemented a normalized relational database schema
* Created tables for students, instructors, courses, lessons, bookings, rooms, and learning materials
* Defined primary keys, foreign keys, constraints, and referential integrity rules
* Developed SQL queries for common business operations
* Implemented PL/pgSQL triggers and stored functions to automate database processes
* Tested the system using sample data and realistic usage scenarios

## Features

* Student enrollment management
* Lesson scheduling and tracking
* Instructor assignment
* Room booking management
* Course material distribution
* Automated validation through database constraints and triggers

## Skills Demonstrated

* Database Design
* Entity Relationship Modeling
* SQL Query Development
* PostgreSQL
* PL/pgSQL Triggers and Functions
* Data Integrity and Normalization
